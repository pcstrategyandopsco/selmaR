# selmaR: R Client for the SELMA Student Management System API

Provides a clean interface to the SELMA student management system REST
API. Handles authentication, Hydra JSON-LD pagination, and data
standardisation. Supports fetching students, enrolments, intakes,
components, and programmes with optional RDS caching. Includes utilities
for NZ/AU phone normalisation, SELMA date parsing, and EFTS pro-rata
calculation for New Zealand TEOs.
